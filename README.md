# AMT Speedometer
## Version 1.0

> Free speedometer for FiveM (Standalone)

## Features

- You can choose speed unit directly on 'client.lua' file (KM/H or MPH)
- Gearbox disabled for Helicopter or Plane
- Easily configurable
